## 简介

## 核心特性

| 特性 | 说明 |
|------|------|
| 高性能 | 与 Node.js 和 Go 相当 |
| 自动文档 | Swagger UI/ReDoc |
| 类型安全 | Pydantic 集成 |
| 异步支持 | async/await |
