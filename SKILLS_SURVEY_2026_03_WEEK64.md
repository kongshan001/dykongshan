## 🔧 六、实战应用建议

### 6.1 游戏客户端开发工作流

```
1. 项目规划 → game-development
2. 引擎选择 → unity/godot/unreal
3. 核心开发 → game-ai, 2d-games/3d-games
4. 测试验证 → game-testing, unity-test-automation
5. 部署发布 → docker, cicd-pipeline
```

### 6.2 Python 开发工作流

```
1. 项目初始化 → python-script-generator
2. API 开发 → FastAPI
3. 异步处理 → async-python-patterns
4. 数据处理 → python-dataviz
5. 测试 → pytest, e2e-testing
6. 部署 → docker, azure-*-py
```

### 6.3 自动化测试工作流

```
1. 测试策略 → test-master
2. E2E 测试 → playwright-skill
3. 移动端测试 → android-adb
4. API 测试 → api-security-testing
5. 持续集成 → github-actions-templates
```
