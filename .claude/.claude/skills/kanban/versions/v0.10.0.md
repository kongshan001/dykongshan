## 概述

本次版本是 kanban 框架的一次重大架构升级，涵盖三个核心方向：
1. Agent 体系规范化（统一 kanban- 前缀 + 权限全能力化）
2. Init 智能化（项目扫描 + 基础设施检测）
3. FSM 流程扩展（Plan Review → QA Spec → Spec Review 质量门禁 + 多 Executor 并行）

## 新增 Agent

| Agent | 职责 |
|-------|------|
| kanban-planner | 需求分析 + 任务拆解（含并行化元数据） |
| kanban-plan-reviewer | Plan 质量审核（6 维度评分） |
| kanban-qa | 测试用例文档产出（spec 模式）+ 代码验收（eval 模式） |
| kanban-test-spec-reviewer | 测试用例文档审核（5 维度评分） |
| kanban-executor | 编码实现 |
| kanban-code-reviewer | 代码审核 |
| kanban-designer | 设计评审 |
| kanban-pm | 产品验收 |
| kanban-researcher | 技术调研 |
| kanban-knowledge-manager | 知识管理 + 复盘 |

## 完整 FSM 流程

```
Plan → Plan Review → QA Spec → Spec Review → Execute → Evaluate → Retrospective → User Decision → Archive
```
