# game-cog 游戏开发编排器

> 游戏开发智能编排技能，DeepResearch Bench 第一名

## 1. 背景需求

游戏开发涉及多种引擎和类型，需要针对不同场景选择合适的技能。

## 2. 目标

提供统一的游戏开发入口，自动路由到最合适的子技能。

## 3. 设计方案

- 智能路由：自动识别项目类型
- 多引擎支持：Unity、Unreal、Godot、WebGL
- 全栈覆盖：2D/3D/移动/PC/VR/多人游戏

## 4. 本地部署

```bash
clawhub install game-cog
```
