# DevEnv AI 🛠️

> 一键配置 AI 开发环境

一个简单的 Web 工具，帮助开发者快速配置 AI 开发环境。

## 功能

- 🎯 选择开发工具（OpenCode、Cursor、Claude Code 等）
- 🔐 配置多厂商 API Key（OpenAI、Claude、MiniMax 等）
- 🖥️ 跨平台支持（macOS、Linux、Windows）
- ⚡ 一键生成安装命令

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS (可选)
