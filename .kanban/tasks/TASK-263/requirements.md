# 需求分析: TASK-263 旅行离线翻译器

## 功能需求
- FR-001: 离线文本翻译（中/英/日/韩/法/西/德/泰等 20+ 语言）
- FR-002: 离线语音翻译（说话即翻译）
- FR-003: 拍照翻译（菜单/路牌/说明书）
- FR-004: 常用旅行短语库（按场景分类：问路/点餐/购物/就医）
- FR-005: 翻译历史记录与收藏

## 非功能需求
- NFR-001: 离线翻译延迟 < 1 秒
- NFR-002: 语言包单个 < 200MB
- NFR-003: OCR 识别准确率 > 90%
- NFR-004: 语音识别支持嘈杂环境

## 验收标准
- AC-001: 无网络环境下完整使用文本/语音/拍照翻译
- AC-002: 语言包下载后自动切换离线模式
- AC-003: 翻译结果可朗读（TTS）
- AC-004: 短语库按场景快速检索
