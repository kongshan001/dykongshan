# 旅行时差管理工具 - 需求文档

## 背景
跨时区旅行者经常受到时差反应(jet lag)困扰，需要科学的方法管理调整。

## 功能需求
1. 时差计算器：输入出发地和目的地，自动计算时差
2. 调整计划生成：根据飞行方向（东行/西行）生成个性化调整方案
3. 作息建议：出发前3天开始的睡眠时间调整建议
4. 褪黑素使用指南：基于科学研究的用量和时间建议
5. 抵达后适应方案：目的地当地时间的活动/光照建议

## 非功能需求
- Markdown格式文档
- 覆盖主要跨时区航线

## 验收标准
- 支持全球主要城市时区
- 包含东行和西行两种调整策略
- 提供出发前和抵达后的完整方案
