# TASK-238 旅行翻译助手 — Plan

## 需求
常用旅行短语多语言翻译，支持中/英/日/法/泰/韩。

## 功能
1. **场景分类** — 问候/交通/餐饮/购物/紧急/住宿
2. **多语言短语库** — 内置6语言常用短语对照表
3. **发音提示** — 中文拼音+罗马音标注
4. **快速查找** — 按场景/关键词检索

## 技术方案
- Python 3.11+, 纯标准库
- 内置短语数据库
- 入口: travel_translate/__main__.py

## 文件结构
```
travel_translate/
├── __main__.py   # CLI入口+demo
├── models.py     # 数据模型
├── phrases.py    # 短语数据库
└── report.py     # 格式化输出
```
