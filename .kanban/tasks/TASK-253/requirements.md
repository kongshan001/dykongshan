# 需求分析: 旅行花费记账本

## 功能需求
- FR-001: 多币种花费记录（金额/类别/时间/地点/备注）
- FR-002: 实时汇率转换，支持 50+ 币种
- FR-003: 按类别/日期/行程维度统计支出
- FR-004: 旅行预算设定与超支预警
- FR-005: 多人旅行分账计算（AA/自定义比例）

## 非功能需求
- NFR-001: 离线记账，上线后同步
- NFR-002: 汇率数据缓存，每日更新
- NFR-003: 支持导出 CSV/PDF 报表
- NFR-004: 记账操作响应 < 500ms

## 技术约束
- 汇率数据源：ExchangeRate-API
- 图表：ECharts
- 端到端加密存储敏感财务数据

## 验收标准
- AC-001: 选择币种后自动显示换算后金额
- AC-002: 统计图表实时更新
- AC-003: 分账结果精确到分
- AC-004: PDF 报表包含完整花费明细
