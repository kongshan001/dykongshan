# TASK-236 旅行美食推荐引擎 — Plan

## 需求
基于目的地和口味偏好推荐当地特色美食和餐厅。

## 功能
1. **美食数据库** — 按城市收录特色美食（菜名/类型/价格/评分/推荐理由）
2. **口味偏好匹配** — 辣/清淡/甜/海鲜/素食/肉食偏好过滤
3. **预算筛选** — 廉价/中等/高档
4. **推荐报告** — 按餐次（早/午/晚/小吃）分类输出

## 技术方案
- Python 3.11+, 纯标准库
- 入口: travel_food/__main__.py

## 文件结构
```
travel_food/
├── __main__.py   # CLI入口+demo
├── models.py     # 数据模型
├── database.py   # 美食数据库
├── matcher.py    # 偏好匹配引擎
└── report.py     # 格式化输出
```
