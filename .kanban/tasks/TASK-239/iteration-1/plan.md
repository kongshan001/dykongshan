# TASK-239 旅行时差计算器 — Plan

## 需求
多时区对比、倒时差建议、最佳联络时间。

## 功能
1. **时区数据库** — 主要城市UTC偏移+夏令时信息
2. **多城市时钟对比** — 同时显示多个城市当前时间
3. **时差计算** — 两城市间时差+方向
4. **倒时差建议** — 根据飞行方向给出调时差建议
5. **最佳联络时间** — 找两地都在合理时段的重叠窗口

## 技术方案
- Python 3.11+, zoneinfo标准库
- 入口: travel_timezone/__main__.py

## 文件结构
```
travel_timezone/
├── __main__.py   # CLI入口+demo
├── models.py     # 数据模型
├── cities.py     # 城市时区数据
├── calculator.py # 时差计算
└── report.py     # 格式化输出
```
