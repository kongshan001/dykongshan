# 需求分析: TASK-260 旅行保险对比器

## 功能需求
- FR-001: 输入旅行目的地/天数/人数，推荐匹配保险产品
- FR-002: 多款保险产品横向对比（保障范围/价格/理赔条件）
- FR-003: 保险条款关键信息提取与通俗化解读
- FR-004: 在线购买跳转与保单管理
- FR-005: 理赔流程指引与材料清单

## 非功能需求
- NFR-001: 对比结果加载 < 2 秒
- NFR-002: 覆盖 20+ 保险公司产品
- NFR-003: 条款解读准确率 > 95%
- NFR-004: 支持中英双语界面

## 验收标准
- AC-001: 输入行程信息后即时展示推荐产品
- AC-002: 对比表格一目了然，高亮差异项
- AC-003: 条款解读使用通俗语言
- AC-004: 理赔材料清单可导出 PDF
