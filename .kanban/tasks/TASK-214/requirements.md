# 旅行交通卡攻略 - 需求文档

## 背景
旅行者需要了解目的地城市的交通卡系统，包括购买方式、充值、适用线路、优惠等。

## 功能需求
1. 提供主要旅行城市的交通卡信息（东京Suica/Pasmo、伦敦Oyster、纽约MetroCard、香港八达通等）
2. 每种卡片包含：购买渠道、价格、充值方式、适用范围、退卡流程
3. 对比分析：游客专属卡 vs 本地交通卡
4. 实用小贴士：如何省钱、常见陷阱

## 非功能需求
- Markdown格式文档，便于阅读和分享
- 信息准确，基于2025-2026年最新政策

## 验收标准
- 覆盖至少10个热门旅行城市的交通卡
- 每种卡片信息完整
- 包含实用建议
