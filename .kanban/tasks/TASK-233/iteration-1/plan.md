# TASK-233 旅行备忘清单生成器 — Plan

## 需求
根据目的地、季节、旅行类型自动生成打包清单和出行备忘。

## 功能设计
1. **目的地感知** — 根据城市气候推荐衣物
2. **季节判断** — 基于出行月份自动判断季节
3. **旅行类型** — 商务/休闲/户外/海岛，不同类型不同清单
4. **分类输出** — 证件/衣物/电子/洗漱/药品/其他

## 技术方案
- Python 3.11+, 纯标准库
- 入口: `travel_checklist/__main__.py`
- 规则引擎：基于目的地+季节+类型的规则匹配

## 文件结构
```
travel_checklist/
├── __main__.py    # CLI入口+demo
├── models.py      # 数据模型
├── rules.py       # 清单规则引擎
├── cities.py      # 城市气候数据
└── report.py      # 格式化输出
```
