# TASK-231 旅行行程优化器 — Plan

## 需求
基于时间约束和偏好，自动优化旅行行程路线和日程安排。

## 功能设计
1. **景点库** — 内置热门城市景点数据（开放时间、建议游玩时长、评分）
2. **时间约束引擎** — 根据可用天数、每日游玩时间生成可行日程
3. **路线优化** — 基于贪心算法安排景点顺序（最短路径近似）
4. **偏好权重** — 支持文化/自然/美食/购物偏好调整

## 技术方案
- Python 3.11+, 纯标准库
- 入口: `travel_planner/__main__.py`
- 路线优化: 贪心最近邻算法

## 文件结构
```
travel_planner/
├── __main__.py    # CLI 入口 + demo
├── models.py      # 数据模型
├── spots.py       # 景点数据库
├── optimizer.py   # 路线优化
├── schedule.py    # 日程生成
└── report.py      # 输出报告
```

## 验收标准
- `python -m travel_planner` 可运行
- 输入城市+天数，输出优化日程表
- 景点按地理距离合理排序
