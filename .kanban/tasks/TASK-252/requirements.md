# 需求分析: 旅行美食地图

## 功能需求
- FR-001: 基于当前位置推荐周边当地特色美食（餐厅/小吃/市场）
- FR-002: 美食打卡记录，支持拍照+评分+点评
- FR-003: 美食地图可视化，标注已打卡和待打卡地点
- FR-004: 按菜系/价格/距离/评分筛选推荐
- FR-005: 旅行美食排行榜，汇总推荐热门美食

## 非功能需求
- NFR-001: 位置推荐响应 < 2 秒
- NFR-002: 支持离线查看已收藏美食
- NFR-003: 地图渲染支持 500+ 标注点
- NFR-004: 图片上传自动压缩 < 300KB

## 技术约束
- 地图使用 Mapbox GL
- 推荐算法基于距离 + 评分 + 个人口味偏好
- 数据源：大众点评/Google Places API

## 验收标准
- AC-001: 打开地图 2 秒内显示周边美食标注
- AC-002: 打卡后数据实时同步到地图
- AC-003: 筛选条件切换即时生效
- AC-004: 可生成旅行美食分享卡片
