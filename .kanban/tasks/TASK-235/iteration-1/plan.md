# TASK-235 旅行安全指南生成器 — Plan

## 需求
根据目的地生成安全须知、紧急联系方式、风险提示。

## 功能
1. **目的地安全数据库** — 治安/卫生/自然灾害/交通安全评级
2. **紧急联系方式** — 当地报警/急救/使馆电话
3. **风险提示** — 常见诈骗/盗窃/健康风险
4. **安全清单** — 出行前安全准备checklist
5. **分类输出** — 按风险类型分组展示

## 技术方案
- Python 3.11+, 纯标准库
- 内置安全数据库（JSON结构）
- 入口: travel_safety/__main__.py

## 文件结构
```
travel_safety/
├── __main__.py   # CLI入口+demo
├── models.py     # 数据模型
├── database.py   # 安全数据库
├── checklist.py  # 安全准备清单
└── report.py     # 格式化输出
```
