# 需求分析: 旅行退税助手

## 功能需求
- FR-001: 查询各国退税政策（税率/起退额/退税方式/所需单据）
- FR-002: 拍照识别退税单，自动提取关键信息
- FR-003: 退税单状态跟踪（待盖章/待邮寄/处理中/已退税）
- FR-004: 退税金额计算器（含手续费/汇率换算）
- FR-005: 退税提醒（登机前提醒盖章/归国后提醒邮寄）

## 非功能需求
- NFR-001: 覆盖 50+ 热门购物国家退税政策
- NFR-002: 退税单 OCR 识别准确率 > 95%
- NFR-003: 离线可查看已保存退税单
- NFR-004: 退税金额计算精确到小数点后两位

## 技术约束
- OCR：Google Vision API / Tesseract
- 退税政策数据库：自建 + 定期更新
- 提醒：FCM 推送 + 本地通知

## 验收标准
- AC-001: 输入国家即时显示退税政策摘要
- AC-002: 拍照退税单后 3 秒内提取关键信息
- AC-003: 退税状态变更时推送通知
- AC-004: 计算器结果与实际退税金额误差 < 1%
