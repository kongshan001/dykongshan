# 需求分析: 旅行纪念品清单

## 功能需求
- FR-001: 创建旅行纪念品清单，支持添加物品名称、价格、购买地点、照片
- FR-002: 拍照识别纪念品，自动填充名称和预估价格
- FR-003: 按旅行行程关联纪念品，地图标注购买地点
- FR-004: 预算管理：设定纪念品预算，实时追踪花费
- FR-005: 分享纪念品清单给朋友（心愿单/推荐单）

## 非功能需求
- NFR-001: 图片识别响应 < 3 秒
- NFR-002: 支持离线添加，上线后同步
- NFR-003: 单个清单支持 200+ 物品
- NFR-004: 图片压缩后存储，单张 < 500KB

## 技术约束
- 图片识别使用 OpenAI Vision API
- 地图标注复用 Mapbox
- 前端 React Native（跨平台）

## 验收标准
- AC-001: 拍照后 3 秒内识别并填充物品信息
- AC-002: 清单内物品总数和总金额实时显示
- AC-003: 预算超支时自动提醒
- AC-004: 可生成分享链接供好友查看
