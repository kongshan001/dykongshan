# 需求分析: 旅行天气穿搭建议

## 功能需求
- FR-001: 获取目的地 7 天天气预报（温度/湿度/风力/降水概率）
- FR-002: 根据天气+活动类型推荐每日穿搭方案
- FR-003: 用户衣橱管理，添加个人衣物（拍照/分类）
- FR-004: 智能搭配：基于衣橱自动生成搭配建议
- FR-005: 穿搭方案分享到社交媒体

## 非功能需求
- NFR-001: 天气数据更新频率 ≤ 30 分钟
- NFR-002: 穿搭推荐响应 < 2 秒
- NFR-003: 离线查看已生成的穿搭方案
- NFR-004: 支持 5 种活动场景（城市观光/海滩/登山/滑雪/正式场合）

## 技术约束
- 天气数据：Open-Meteo API（免费）
- 穿搭推荐规则引擎 + LLM 辅助生成
- 衣橱图片存储：S3 + CDN

## 验收标准
- AC-001: 输入目的地即显示 7 天天气 + 穿搭建议
- AC-002: 穿搭建议基于用户实际衣橱衣物
- AC-003: 可手动调整推荐方案并保存
- AC-004: 生成的穿搭卡片可一键分享
