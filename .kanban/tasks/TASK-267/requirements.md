# 需求分析: TASK-267 旅行充电器适配查询

## 功能需求
- FR-001: 输入目的地查询该国插座类型/电压/频率
- FR-002: 根据用户设备推荐所需转换器/适配器
- FR-003: 全球插座类型图鉴（A~O 型图文对照）
- FR-004: 附近购买转换器的商店/便利店定位
- FR-005: 多目的地行程的充电准备清单

## 非功能需求
- NFR-001: 数据覆盖 200+ 国家/地区
- NFR-002: 查询响应 < 1 秒
- NFR-003: 离线可查询已缓存国家
- NFR-004: 插座图鉴支持缩放查看

## 验收标准
- AC-001: 选择国家即时显示插座信息
- AC-002: 根据出发地和目的地对比差异
- AC-003: 图鉴清晰标注插脚形状和尺寸
- AC-004: 多国行程生成完整充电准备清单
