# 需求文档: 旅行预算追踪与费用分类

## 1. 背景需求
旅行中需要实时追踪花费，按类别统计预算执行情况，避免超支。

## 2. 功能目标
- 多币种费用记录，自动汇率转换
- 费用分类：交通、住宿、餐饮、门票、购物、其他
- 每日/总预算对比，超支预警
- 费用分享（AA制分账）

## 3. 技术方案
- 后端：费用记录CRUD + 汇率API集成
- 实时统计聚合（按天/按类别/按总预算）
- 分账算法（支持不等额分摊）

## 4. 验收标准
- [ ] 支持至少10种货币
- [ ] 费用记录响应 < 200ms
- [ ] 预算超支实时推送通知
- [ ] 分账结果精确到分
