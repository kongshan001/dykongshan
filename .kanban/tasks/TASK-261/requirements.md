# 需求分析: TASK-261 旅行汇率提醒

## 功能需求
- FR-001: 设定目标汇率，到达时推送提醒
- FR-002: 实时汇率展示（多币种看板）
- FR-003: 汇率走势图（7天/30天/90天）
- FR-004: 最佳换汇时机建议（基于历史趋势分析）
- FR-005: 附近换汇点/ATM 定位与费率对比

## 非功能需求
- NFR-001: 汇率数据延迟 < 30 秒
- NFR-002: 支持 100+ 币种
- NFR-003: 提醒推送到达率 > 99%
- NFR-004: 离线查看最近缓存汇率

## 验收标准
- AC-001: 设定汇率提醒后精确触发
- AC-002: 走势图交互流畅，支持缩放
- AC-003: 换汇建议附数据依据
- AC-004: 附近换汇点距离 < 100m 精度
