# 旅行宠物友好指南

## 亚洲
### 日本
- 入境: 需芯片植入、狂犬疫苗证明、健康证书，提前40天申请
- 航空: ANA/JAL 允许小型宠物客舱携带（限重8kg含笼）
- 宠物友好酒店: 多数商务酒店禁止，推荐 Search 网站筛选

### 泰国
- 入境: 需出口国官方健康证明、狂犬疫苗
- 宠物友好: 苏梅岛/普吉岛部分度假村允许

## 欧洲
### 法国
- 入境: EU宠物护照或健康证明+芯片+狂犬疫苗
- 宠物友好: 高度友好，餐厅/公共交通普遍允许小型宠物

## 美洲
### 美国
- 入境: CDC要求犬猫健康证明，部分州有额外要求
- 航空: 各航空公司政策不同，情感支持动物政策已收紧

## 准备清单
- [ ] 植入ISO标准芯片
- [ ] 狂犬疫苗接种证明（21天以上）
- [ ] 官方健康证书
- [ ] 航空公司宠物舱位预订
- [ ] 旅行保险（含宠物）
