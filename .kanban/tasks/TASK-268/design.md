# TASK-268 设计文档: 旅行WiFi热点地图

## 技术栈选型
- **前端**: React 18 + TypeScript + Leaflet.js (地图)
- **后端**: FastAPI + PostgreSQL + PostGIS
- **离线**: Service Worker + IndexedDB
- **部署**: Docker Compose

## 核心功能清单
1. 地图展示附近免费WiFi热点(咖啡馆/图书馆/商场/机场)
2. 热点详情(网络名/速度评级/密码/使用限制)
3. WiFi速度测试与用户评分
4. 离线热点地图下载与无网导航
5. 自动连接已保存热点(密码管理)

## 验收标准
- AC-001: 地图即时显示周边WiFi热点标注
- AC-002: 速度测试10秒内完成
- AC-003: 离线模式可导航至已缓存热点
- AC-004: 密码自动填充连接成功

## 约束条件
- 热点数据库覆盖100+城市
- 地图加载<3秒
- 离线地图支持500+热点
- 速度测试误差<10%
- 移动端优先响应式设计
