# 需求分析: 旅行紧急求助卡片

## 功能需求
- FR-001: 自动生成目的地紧急求助卡片（报警/急救/消防电话）
- FR-002: 显示最近大使馆/领事馆地址和联系方式
- FR-003: 附近医院/药房定位与导航
- FR-004: 一键拨打当地紧急电话（支持 SOS 按钮）
- FR-005: 多语言紧急短语翻译（我需要帮助/我受伤了等）

## 非功能需求
- NFR-001: 求助卡片支持离线访问
- NFR-002: 数据覆盖 200+ 国家/地区
- NFR-003: SOS 按钮响应 < 1 秒
- NFR-004: 紧急短语支持 20+ 语言

## 技术约束
- 紧急电话数据源：Wikipedia + 各国官方数据
- 大使馆数据：Google Places API
- 翻译：离线 TTS + 预置翻译库

## 验收标准
- AC-001: 选定目的地后即时生成求助卡片
- AC-002: SOS 按钮可直接拨号（移动端）
- AC-003: 离线状态下仍可查看卡片内容
- AC-004: 紧急短语播放清晰准确
