# 需求分析: TASK-264 旅行投票决策器

## 功能需求
- FR-001: 创建投票议题（目的地/餐厅/活动/住宿），邀请同行人参与
- FR-002: 多种投票模式（单选/多选/评分/排名）
- FR-003: 实时投票结果可视化（饼图/柱状图）
- FR-004: 投票截止时间设置，到期自动公布结果
- FR-005: 匿名投票选项，避免从众心理

## 非功能需求
- NFR-001: 投票响应 < 500ms
- NFR-002: 支持 20 人同时投票
- NFR-003: 实时结果更新延迟 < 2 秒
- NFR-004: 支持离线投票，上线后同步

## 验收标准
- AC-001: 创建投票后同行人即时收到通知
- AC-002: 投票结果图表实时更新
- AC-003: 截止后自动推送最终结果
- AC-004: 匿名模式下无法查看他人选择
