# 需求分析: 旅行时差调节器

## 功能需求
- FR-001: 输入出发地和目的地，自动计算时差
- FR-002: 基于科学方法生成个性化时差调节计划（光照/睡眠/饮食时间调整）
- FR-003: 出发前 N 天开始提醒执行调节步骤
- FR-004: 实时跟踪调节进度，动态调整建议
- FR-005: 提供时差适应期间的咖啡因/褪黑素使用建议

## 非功能需求
- NFR-001: 支持任意两时区组合
- NFR-002: 调节计划生成 < 1 秒
- NFR-003: 提醒准时率 > 99%
- NFR-004: 计划可导出为日历事件

## 技术约束
- 时差调节算法基于光照疗法研究论文
- 时区数据：IANA Time Zone Database
- 日历集成：Google Calendar / Apple Calendar

## 验收标准
- AC-001: 输入两地即时显示时差和调节建议
- AC-002: 调节计划按天展示，含具体作息时间表
- AC-003: 提醒按计划准时推送
- AC-004: 可一键导入日历
