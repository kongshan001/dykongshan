# 需求文档: 旅行行程PDF导出功能

## 1. 背景需求
旅行规划应用需要一个将行程导出为PDF的功能，方便用户离线查看和分享旅行计划。

## 2. 功能目标
- 将完整旅行行程（含日期、地点、交通、住宿、活动）导出为格式美观的PDF
- 支持自定义封面（旅行标题、日期范围、目的地图片）
- 支持按天/按城市分章节
- 包含预算汇总表
- 输出A4尺寸，适合打印

## 3. 用户故事
- 作为一个旅行者，我想导出PDF行程单，以便离线查看旅行安排
- 作为一个旅行者，我想在PDF中看到预算汇总，以便控制旅行花费

## 4. 技术方案
- 前端使用 HTML 模板 + Puppeteer/Playwright 生成PDF
- 后端提供 API: POST /api/trips/:id/export-pdf
- 支持异步生成，大行程排队处理
- PDF模板支持自定义主题色

## 5. 验收标准
- [ ] 能成功导出包含3天以上行程的PDF
- [ ] PDF包含封面、行程详情、预算汇总
- [ ] 导出时间 < 10秒（10天以内行程）
- [ ] 中文内容正确渲染，无乱码
