# TASK-302 设计文档: 旅行签证进度追踪

## 技术栈选型
- **前端**: React 18 + TypeScript + Ant Design
- **后端**: FastAPI + SQLite + APScheduler
- **通知**: Web Push + Email SMTP

## 核心功能清单
1. 签证申请进度查询(支持多国)
2. 申请材料清单管理(可自定义)
3. 签证到期提醒(30/14/7天通知)
4. 材料上传与归档
5. 申请状态时间线展示

## 验收标准
- AC-001: 支持至少10国签证进度查询
- AC-002: 到期提醒准确触发
- AC-003: 材料清单可自定义
