# 需求文档: 旅行多语言实时翻译

## 1. 背景需求
海外旅行语言不通是最大障碍，需要实时翻译工具辅助沟通。

## 2. 功能目标
- 文字/语音实时翻译，支持50+语言
- 拍照翻译（菜单/路牌/说明）
- 常用旅行短语库（离线可用）
- 对话模式（双方交替说话自动翻译）

## 3. 技术方案
- 集成翻译API（Google Translate / DeepL）
- OCR拍照翻译（Tesseract + 翻译API）
- 离线短语库SQLite存储
- WebSocket实时语音流

## 4. 验收标准
- [ ] 文字翻译延迟 < 500ms
- [ ] 语音翻译延迟 < 2秒
- [ ] 拍照翻译准确率 > 85%
- [ ] 离线短语库覆盖10种语言
