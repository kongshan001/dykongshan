## 简介

Playwright 是一个强大的浏览器自动化和 E2E 测试技能，支持主流浏览器自动化。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 多浏览器支持 | Chromium、Firefox、WebKit |
| 跨平台 | Windows、macOS、Linux |
| API 测试 | REST、GraphQL |
| 移动端模拟 | iOS、Android |

---

## 游戏客户端测试

Playwright 特别适合 Web/H5 游戏测试：

- UI 自动化测试
- 游戏登录流程
- 资源加载验证
- 性能基准测试
